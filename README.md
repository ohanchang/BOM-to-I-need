# BOM to I need — 批次格式整理與成本拆解分頁重構工具

純前端 BOM 批次整理與多工作表重構工具。整合了格式對齊、子料行移位修正、美金匯率換算、料號分類與大分類、物料拆解分析的跨表 Excel 公式重構，並打包為 ZIP 下載。

---

## 🚀 專案特點

- **多 BOM 批次處理**：可一次拖放或選取複數份 BOM 檔案進行批次處理，支援大檔案極速處理。
- **類型識別與略過**：自動偵測上傳的 BOM 檔案是「原始 BOM」還是「已整理的 BOM」，並提供一鍵「略過已整理的 BOM」之篩選打包下載。
- **排版與偏移修正**：自動將偏離的標題列（應為 Row 12/0-indexed Row 11）及錯位的子料行欄位（應為 Column 9-12 移位至 11-14 的項目）移回標準格式。
- **「乘積」欄位插入**：在 Column L (12 欄) 插入乘積欄位，公式設為 `=IFERROR(使用數量 * 單價, 0)`。
- **分類分頁拆分 (核心功能)**：
  - 將原本混雜在主要 BOM 表右側的摘要分析區塊徹底剝離。
  - **分頁 1 (BOM)**：簡潔標準的 BOM 資料明細與自動色彩高亮標記（無單價高亮為黃色，PCB/線材/8B/4W/機構件特定字型顏色）。
  - **分頁 2 (大分類統計)**：自動生成 1xxx~8xxx 的 8 大類統計，帶有標準的跨表 `SUMIFS` 函數。
  - **分頁 3 (物料拆解分析)**：自動生成 13 大類物料成本拆解，配備與 VBA 完全相同的跨工作表 `SUMPRODUCT` 公式。
- **跨 BOM 智慧單價推薦**：上傳多份 BOM 後，系統自動掃描缺主料並跨 BOM 比對，提供「精確」或「相似（前綴匹配 ≥5 碼）」推薦單價，並於推薦料號欄位以**藍色高亮共同前綴**。
- **一鍵套用推薦單價**：點擊橙色「⚡ 一鍵套用推薦單價」按鈕可批次填入所有智慧推薦單價並即時重算。
- **完美跨平台公式顯示**：所有生成公式均在 JS 中完成預計算並填入 Cell 的 `v` 屬性，確保不支援公式計算的檢視器（如手機或預覽版）也能正常顯示，同時保留 active Excel 變更連動公式。
- **本機化與離線運作**：核心程式庫 (`xlsx.full.min.js`, `jszip.min.js`) 完全本機化，支援完全無網路環境下百分之百離線執行。

## 📋 使用說明

1. 開啟 `index.html`。
2. 設定 **全域美金匯率** (預設 30)；若上傳檔案包含 `Ctrl Q` 工作表，系統將自動從中讀取匯率。
3. 勾選或取消勾選 **「略過已整理的 BOM 檔案」**。
4. 拖放或點擊上傳多個 BOM 檔案（將自動執行 BOM 格式修正、價格計算、用料比例分析、缺料對比）。
5. 瀏覽處理清單：
   - 點擊 **「展開整理明細與網頁預覽」** 可直接在網頁上切換預覽 `BOM 資料表`、`大分類統計`、`物料拆解分析` 三個分頁，甚至包括欄位移位的 `修正日誌`。
   - 若有缺主料，分頁列表會出現 **「缺主料價格分析 (N)」** 分頁，顯示缺料清單、推薦料號（含藍色前綴高亮）及推薦單價。
   - 標題列旁出現 **「⚡ 一鍵套用推薦單價」** 橙色按鈕時，可一鍵批次填入所有推薦單價並即時重算。
6. 點擊 **「一鍵打包下載 (ZIP)」** 即可獲取處理完畢的 Excel 檔案壓縮包（含補登價格）。
